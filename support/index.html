Is there some type of guide I can read about color contrast?

Yes, we wrote one. It's right here.

Do I really to need check the colors I use in my designs?

You don't have to check all of your colors, only ones you want people to be able to read. If you're uncertain about all this, definitely read the guide.

Any plans for quick keys or other features?

Totally! We've got a long list, but wanted to ship this with critical features only. We're quick key junkies too and this is a high priority.

Does the app window persist even when clicking on other apps?

By default it only persists when using the eyedropper tool. If you'd like it to always be visible no matter what, you can drag it off of the menu bar to anywhere on the screen.

Can I disable the sounds?

Yes. Click the settings icon on the far right and deselect Sounds.

Can I color pick anwhere on my screen?

For sure! Click and pick away, my friend. Anwhere on the screen your little heart desires. ‚ù§Ô∏è

Does this work inside my design tool?

Yes! We've tested colors in Sketch, Figma, XD, Illustrator, and *Photoshop.

*Photoshop's color space needs to be set to Display to have the same color space as Contrast.

Can I adjust my hex code after choosing it?

Sure can. With either of the fields selected you can use arrow key up or arrow key down to adjust the brightness of your color.
